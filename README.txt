<strong>INSTALL</strong>
1. Upload the Fabricated child theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Fabricated theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

<strong>WIDGET AREAS</strong>
Primary Sidebar - This is the primary sidebar if you are using the Content/Sidebar or Sidebar/Content Site Layout option.
Home - Slider - This is the slider section on the home page.
Home - Top - Area for highlighting featured pages, posts or categories.
Home - CTA - This is the call to action section on the home page.
Home - Middle - Area for highlighting featured pages, posts or categories.
Home - Twitter - Specifically designed to highlight your latest tweet.
Footer #1 - This is the first column of the footer section.
Footer #2 - This is the second column of the footer section.
Footer #3 - This is the third column of the footer section.

<strong>FEATURED IMAGES</strong>
By default WordPress will create a default thumbnail image for each image you upload and the size can be specified in your dashboard under Settings > Media. In addition, the Executive child theme creates the following thumbnail images you'll see below, which are defined (and can be modified) in the functions.php file. These are the recommended thumbnail sizes that are used on the Fabricated child theme demo site.

featured - 285px by 100px
slider - 1140px by 445px
